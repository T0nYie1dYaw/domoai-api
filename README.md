DomoAI API
===
**Unofficial** DomoAI API

Feature
---

- Support all AI commands of DomoAI.
    + `/video`
    + `/move`
    + `/animate`
    + `/gen`
    + `/real`
- Support `upscale` and `vary`
- Streamlit Demo provided
