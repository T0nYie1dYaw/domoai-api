DomoAI API
===
**非官方** DomoAI API.

[English Documentation](README.md)

特性
---

1. 支持全部 [DomoAI](https://domoai.app/) 的AI指令.
    + `/video`
    + `/move`
    + `/animate`
    + `/gen`
    + `/real`
2. 支持 `upscale`(放大) 和 `vary`(变体)
3. 支持使用 `FAST`(高速)模式 和 `RELAX`(休闲)模式
4. 支持查询任务状态
5. 支持事件回调
6. 支持 Docker 部署
7. 提供基于 Streamlit 的示例
